# Criador de jogos e estatística dos jogos da loteria CAIXA. 
API usada: https://github.com/guto-alves/loterias-api

*Um mini projeto para testar habilidades de POO e Python.*

É possível atraves da API e do programa, ver os últimos concursos dos jogos mais famosos da CAIXA. Também leves estatisticas como a quanto tempo um determinado número não é sorteado, para obter mais chances.
